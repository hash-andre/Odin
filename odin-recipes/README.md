# Odin Recipes

A simple HTML website that links to different recipe pages.

This project was built as part of The Odin Project Foundations course. It includes a main index page and three recipe pages: Lasagna, Pizza, and Tiramisu.

## Skills practiced

- HTML boilerplate
- Headings and paragraphs
- Images
- Links
- Ordered and unordered lists
- File and folder organization

## Future improvements

I would like to add CSS, improve the layout, and make the website responsive.
