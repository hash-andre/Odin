# Odin Recipes

A simple HTML and CSS website that links to different recipe pages.

This project was built as part of The Odin Project Foundations course. It includes a main index page and three recipe pages: Lasagna, Pizza, and Tiramisu.

The website now includes basic CSS styling, custom fonts, colors, images, styled links, decorative elements, and a layout that works well on different screen sizes.

## Pages

* Index
* Lasagna
* Pizza
* Tiramisu

## Skills practiced

* HTML boilerplate
* Semantic page structure
* Headings and paragraphs
* Images and relative paths
* Links between pages
* Ordered and unordered lists
* File and folder organization
* CSS selectors
* Class selectors
* Styling text, links, images, and lists
* Box model: margin, padding, and border
* Basic responsive layout
* Pseudo-elements
* Custom fonts with Google Fonts

## What I learned

During this project, I learned how to build a basic multi-page website and style it with CSS. I practiced organizing files, linking pages together, using relative paths, and improving the visual design of a simple website.

I also started to understand how CSS affects layout, spacing, typography, and readability.
