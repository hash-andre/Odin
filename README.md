# Odin

Projects built while studying [The Odin Project](https://www.theodinproject.com/) Foundations course.

Each project has its own folder and README with more details about the goals, structure, and skills practiced.

## Projects

### 1. [Odin Recipes](./odin-recipes)

A simple multi-page recipe website built with HTML and CSS.

It includes an index page and three recipe pages: Lasagna, Pizza, and Tiramisu.

Main topics practiced:

* HTML structure
* Links and images
* Lists
* File organization
* CSS basics
* Box model
* Basic responsive layout

### 2. [Landing Page](./landing-page)

A responsive landing page built with HTML and CSS.

The project focuses on creating a complete webpage layout using Flexbox and responsive design techniques.

Main topics practiced:

* Flexbox
* CSS custom properties
* Media queries
* Responsive layout
* Image handling
* `min-width` and `min-height`
* Project documentation
