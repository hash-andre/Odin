# OwlWatch Landing Page

A responsive landing page built as part of The Odin Project Foundations course.

The project is themed around owls and focuses on building a complete landing page layout using HTML and CSS.

## Features

* Responsive landing page structure
* Header with navigation
* Hero section with image and call-to-action
* Information cards
* Quote section
* Call-to-action section
* Footer
* Custom owl-themed content and images

## What I Learned

During this project, I practiced several important HTML and CSS concepts:

* Building a complete page structure with semantic HTML
* Using Flexbox to align and distribute elements
* Creating responsive layouts with media queries
* Using `:root` and CSS custom properties for reusable values
* Understanding `min-width` and `min-height`
* Managing page height with `min-height: 100vh`
* Keeping the footer at the bottom of the page when content is short
* Making images responsive with `max-width`, `height: auto`, and `object-fit`
* Cropping and preparing images with GIMP
* Organizing project assets in an `images/` folder

## Technologies Used

* HTML
* CSS
* Flexbox
* Media queries
* GIMP for image cropping
